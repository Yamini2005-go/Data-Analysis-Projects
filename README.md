## 📁 Project 3: Monthly Gold & Silver Rates India (2020-2026)
- This dataset provides a 6-year simulated timeline (2020–2026) of monthly gold and silver prices in India, specifically designed for practicing time-series      forecasting. It serves as an educational bridge for beginners to master predictive models like ARIMA and regression using pre-normalized financial data.

- Sources: Python to generate simulated data

- Inspiration: This is a great touch. It shows you are part of the learning community.

- Date Description: ->The recording date of the metal price. ->"Data is recorded on the 13th of every month from February 2020 to February 2026.

- Rate Description: ->The simulated market price of the precious metal in Indian Rupees (INR). ->"For Gold, this represents the rate per 100 grams. For Silver, this represents the rate per 1 kg. This value includes simulated market volatility."

- Normalized_rate (or new_rate) Description: ->The feature-engineered rate used for Machine Learning model fitting. ->"Calculated as Rate / 100. This column is pre-processed to assist beginners in training neural networks or regression models without needing additional scaling steps."
